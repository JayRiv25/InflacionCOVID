# InflacionCOVID
El código busca reflejar el impacto, aumento y disminución de la inflación en el mundo durante la pandemia de COVID-19 entre los años 2020-2024 mediante interpretaciones gráficas, mapa de calor y distribuciones que evidencien el impacto provocado y a quiénes afectó más a lo largo del mundo.

Dataset con datos referentes a la inflación global post COVID 2020-2024.
URL Dataset: https://www.kaggle.com/datasets/ssssws/global-inflation-dynamics-post-covid-20202024/data
